# Skyblock Data Alpha
![](https://img.shields.io/github/repo-size/peter-hunt/skyblock-data)
![](https://img.shields.io/github/license/peter-hunt/skyblock-data)

Contains data used by [Skyblock](https://github.com/peter-hunt/skyblock).
